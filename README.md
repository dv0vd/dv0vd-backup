# Dv0vD Backup server
Backup server for dv0vd website.

## Getting started
1) Update packages index: `apt update`.
2) Install git: `apt install git`.
3) Clone repo: `git clone https://github.com/dv0vd/dv0vd-backup.git`.
4) Go to the project directory: `cd ./dv0vd-backup`.
5) Configure the `.env` file.
6) Run the initialization script `./deployment/init.sh`.
